Templatemela Theme Settings

You will found Templatemela theme settings into Appearance Menu with name TM Theme Settings



General Settings

  • Logo : It's an easy way to upload logo from Logo Image option also enter logo alter text.
  • Show Control Panel : If you make this option to "yes" then you can see control panel option on front page.
  • Do you want Responsive Design : If you make this option to "yes" then you can see your website set in any size of media
  • Background/Patterns : You can select any background patterns it will display as main background pattern Or you can upload your own background image.
  • Body Font Color : You can set Font color of Body
  • Body Font : You can set Font Family for Body.
  • Buttons Color : You can set background color of Button
  • Buttons Hover Color : You can set background color of Button's Hover

1 : Upload Logo


  • First goto general settings -> click upload logo button -> select your logo image -> click on file URL button -> click on insert into post ->save settings.

Topbar Settings


  • Show Top bar : You can show or hide by setting it as Yes/No.
  • Top Menu background color : You can set background color of Navigation Bar.
  • Top Header Text color : You can set Text color of Top Bar.
  • Top Header Link color : You can set Link color of Top Bar.
  • Top Header Link hover color : You can set Link hover color of Top Bar.
  • Top Menu text color : You can set font color of Navigation text.
  • Top Menu text hover color : You can set font's hover color of Navigation text.
  • Show Contact Information : You can show or hide by setting it as Yes/No.
  • Show Social links : You can show or hide by setting it as Yes/No.

Header Settings


  • Header Background Color : You can set Header Background image, Header Background color.

Content Settings


  • Colors : You can set Heading color (h1 to h6), Link color & Link Hover color.
  • Font : You can set Heading font (h1 to h6).
  • Show custom banner : You can show or hide by setting it as Yes/No.

Footer Settings

  • Footer Background color : You can set Footer Background color.
  • Footer Link Color : You can set Footer Link color.
  • Footer Link Hover Color : You can set Footer Link's hover color.
  • Footer copyright : You have to easy enter the copyright Text from here. You just write down text whatever you want.
  • Footer copyright Link : You can set footer copyright link.

Shop Settings

  • Products per Shop page : You can set products per shop page.
  • All Products per row : You can set how many products to show per row on shop page.
  • Related Products : You can set number of related products.
  • Upsells Products: You can set number of upsells products.

Slider Settings

  • Display slider navigation arrows? : You can show or hide the slider navigation arrow by setting it as Yes/No.
  • Display slider pagination? : You can show or hide the slider pagiantion by setting it as Yes/No.
  • Display Slide Animation type? : You can set the Slide Animation type by setting it as Slide/Fade.
  • Auto play slider? : You can on or off the autoplay feature.
  • Slideshow Speed : You can set the slide show speed in miliseconds eg 7000
  • Animation Duration : You can set the interval time between slide show in miliseconds eg 600

Shortcodes

We Provide You different shortcodes to add from Admin Panel.

Theme Shortcodes

Portfolio

To display portfolio Filter, write this shortcode.

[tm_portfolio_filter posts_per_page="8"]


To display portfolio single category wise, write this shortcode.

[tm_portfolio cat="Portfolio2" column=2 max="4" character="50"]


Services

To display services, write this shortcode.

[service][/service]

Parameters:

color="FFFFFF"

icon_background_color="e29805"

icon="fa-arrows-alt"

title="Your Text"

description=""

link_text="Read More"

link_url="#"

style="1 / 2 / 3"

To display the icon in round, you need to write the "icon_class" attribute which you can get from following link:

http://fontawesome.io/icons/

Copy the text of whatever icon you want to display as "icon". Ex, icon="fa-bell"

Ex. [service style="1" icon="fa-html5" icon_background_color="9f8b6a" color="FFFFFF" title="HTML5 + CSS3"]Meta komentofrazo ci cis, negativa antaumetado la oni, havi frida aga ac.[/service]

Title

To display title, write this shortcode.


[title][/title]

Parameters:

size="small / medium / big"

color="000000"

align="left / center / right"

Ex. [title size="small" align="left"]It is a demo text.[/title]

Gallery

If you want to display gallery column wise, write this shortcode. You can also set size as large, medium, full.


[gallery]

Parameters:

columns="2 / 3 / 4 / 5 "

size="large / thumbnail / small"

ids="IDS of the media you want to display"

Ex. [gallery columns="2" ids="3406,3405,3410,3399"]
